SQL projects

As a data analyst, I work with large amounts of data stored in databases. While spreadsheets are an effective analysis tool, there is a limit to how much data you can handle in spreadsheets. SQL extracts data from databases and greatly simplifies data analysis from bug data. I can use SQL to access multiple tables from a database at once, clean large datasets, and search for insights.

I can export the results of SQL queries to another software for further analysis, such as Tableau (for better visualization), R (for deeper analysis), or even spreadsheets. It is also great for collaborating with other users. There are different SQL dialects and I am most comfortable with Standard SQL using Google BigQuery and T-SQL using Microsoft SQL Server.
